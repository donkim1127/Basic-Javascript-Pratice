# Basic-Javascript-Pratice

This is a small project that involves mainly HTML and Javascript. It is a single web page that several different components:
- A login form where upon submitting a username the user will be greeted by the page.
- A clock that tells the current user's time.
- A form that allows for the user to enter items into a to-do list where the user can also delete individual items from the list.
- A random quote and its author will be displayed onto the page.
- A random image will be displayed onto the page.
- The user's current weather will be displayed onto the page.
